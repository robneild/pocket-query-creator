Mit dieser App kannst Du Pocket Queries erstellen und herunterladen. Pocket Queries sind eine Premium Members Funktion von Geocaching.com.

Man kann mit dieser App nichts anderes tun als auch auf der Geocaching.com Website. Sie dient nur als Komfort auf dem Smartphone.

Mit dieser App alleine kannst Du nicht geocachen. Du brauchst noch eine App, wie z.B. Locus Pro, c:geo oder die offizielle Geocaching App

Wie erstellt man eine Pocket Query

  • Deine geocaching.com Zugangsdaten müssen in den Einstellungen hinterlegt sein
  • Drücke auf
  • Stelle alle Optionen nach deinen Wünschen ein
  • Drücke nochmal auf , um die Pocket Query zu erstellen

Wie lade ich eine Pocket Query herunter

  • Drücke auf
  • Eine Liste mit herunterladbaren Pocket Queries wird angezeigt
  • Wähle eine aus und drücke auf , um den Download zu starten

Importieren der Pocket Query in c:geo

c:geo kann gezippte Pocket Queries importieren, wenn sie nach einem bestimmt Schema benannt und an entsprechender Stelle im Dateisystem gespeichert sind.
  • name - Die Pocket Query Zip-Datei muss mit Zahlen beginnen. Aktiviere 'c:geo Kompatibilität' in den Einstellungen, um dies Sicher zu stellen.
  • location - Am besten speichert man die Pocket Query im c:geo 'gpx' Import Verzeichnis (konfigurierbar).

Hinweise

  • Wenn die Pocket Query erstellt ist, kann es eine Weile dauern, bis sie ausgeführt wird.
  • Es gibt keine Limitierung bei der Anzahl der Pocket Queries, aber es werden maximal 5 in 24 Stunden ausgeführt.
  • Das Ergebnis der Pocket Query wird an die Email-Adresse gesendet, die im Geocaching.com Profil hinterlegt ist.
  • Das Erstellen der Pocket Query kann bei langsamen Verbindungen ein paar Minuten dauern.

FAQ

Q. Was kann ich mit der Zip-Datei tun?
A. Du kannst sie in einer Geocaching App, wie z.B. Locus Pro, c:geo, CacheBox oder Locus Pro importieren.

Q. Wie kann ich die Pocket Query aus dem Email-Anhang extrahieren?
A. Mit einer Email-App (z.B. Gmail oder K-9 Mail) oder einfach über diese App herunterladen.

Q. Kann ich die Liste der Pocket Queries wieder löschen?
A. Nicht mit dieser App, aber auf der Webseite von geocaching.com.
